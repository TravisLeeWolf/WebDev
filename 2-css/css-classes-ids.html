Internal CSS with classes

When you assign a HTML element to a class it allows you to target that class an change only its CSS

This is a paragraph with the first class, in red and 25% bigger.

This is a paragraph with the second class, in blue and 25% smaller.

In this case we are using an id and not a class so the code for CSS is different, see comments.

IDs are used once in a single page but classes can be used multiple times, hence in the paragraphs above it's better to use classes.

Lastly we can use both the class and span to target some text within a paragraph, here to add additional CSS like underline.
